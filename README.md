## 运行
+ $ cd umi-antd-dva-admin
+ $ npm install 或 cnpm install 或 yarn install
+ $ npm start

## 构建
+ $ cd umi-antd-dva-admin
+ $ npm run bulid
+ $ cd dist
+ $ cpnm install live-server -g 或 yarn add global live-server
+ $ live-server (任意web容器启动均可)
